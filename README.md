# GenTrained - OpenGL Game
> Created by AndrewKitaev
> in beta

### How to build?
Linux/Unix
```bash
mkdir build
cd build
cmake ../
cmake --build ../ --config Release
```
Windows
```
In dev
```

### Changing settings
Edit src/game_config.h
```cpp

#define GAME_TITLE "Gen Trained" // Game title
#define GAME_VERSION "v0.13" // Game version

```
